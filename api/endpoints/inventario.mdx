## Products

### List All Products

<RequestExample>
```bash cURL
curl -X GET https://api.fluxora.com/productos \
  -H "Authorization: Bearer {token}"
```
</RequestExample>

**GET** `/productos`

Retrieves all products in the inventory.

#### Response

Returns an array of ProductoDTO objects.

<ResponseExample>
```json
[
  {
    "id": 1,
    "nombre": "Pan Corriente",
    "descripcion": "Pan tradicional",
    "unidadMedida": "kg"
  }
]
```
</ResponseExample>

#### Status Codes

- `200 OK` - Products retrieved successfully

---

## Product Batches

### Create Product Batch

<RequestExample>
```bash cURL
curl -X POST https://api.fluxora.com/productos/1/lotes \
  -H "Authorization: Bearer {token}" \
  -H "Content-Type: application/json" \
  -d '{
    "fechaProduccion": "2024-03-15",
    "fechaVencimiento": "2024-03-22",
    "cantidadProducida": 100,
    "cantidadDisponible": 100
  }'
```
</RequestExample>

**POST** `/productos/{productoId}/lotes`

Creates a new batch for a product.

<Note>
  Requires **ADMIN** or **PRODUCER** role authorization.
</Note>

#### Path Parameters

<ParamField path="productoId" type="long" required>
  Product ID
</ParamField>

#### Request Body

<ParamField body="fechaProduccion" type="LocalDate" required>
  Production date (YYYY-MM-DD)
</ParamField>

<ParamField body="fechaVencimiento" type="LocalDate" required>
  Expiration date (YYYY-MM-DD)
</ParamField>

<ParamField body="cantidadProducida" type="integer" required>
  Quantity produced
</ParamField>

<ParamField body="cantidadDisponible" type="integer" required>
  Available quantity
</ParamField>

#### Status Codes

- `200 OK` - Batch created successfully
- `403 Forbidden` - Insufficient permissions

---

### Deduct Stock

<RequestExample>
```bash cURL
curl -X PUT https://api.fluxora.com/productos/1/descontar-stock \
  -H "Authorization: Bearer {token}" \
  -H "Content-Type: application/json" \
  -d '{
    "cantidad": 50,
    "motivo": "venta"
  }'
```
</RequestExample>

**PUT** `/productos/{productoId}/descontar-stock`

Deducts stock from available batches using FIFO (First In, First Out) method.

#### Path Parameters

<ParamField path="productoId" type="long" required>
  Product ID
</ParamField>

#### Request Body

Accepts a Map with stock deduction details.

#### Status Codes

- `200 OK` - Stock deducted successfully
- `400 Bad Request` - Insufficient stock
