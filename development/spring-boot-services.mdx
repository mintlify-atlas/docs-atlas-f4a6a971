## Overview

Fluxora microservices are built with **Spring Boot 3.5.4** and follow modern Java 21 development patterns. Each service is independently deployable and communicates via REST APIs.

## Application Bootstrap

### Main Application Class

Each microservice has a main application class with key annotations:

```java microservice-cliente/src/main/java/com/microservice/cliente/MicroserviceClienteApplication.java
package com.microservice.cliente;

import org.springframework.boot.SpringApplication;
import org.springframework.boot.autoconfigure.SpringBootApplication;
import org.springframework.cloud.client.discovery.EnableDiscoveryClient;
import org.springframework.cloud.openfeign.EnableFeignClients;

@SpringBootApplication        // Auto-configuration and component scanning
@EnableFeignClients          // Enable OpenFeign declarative REST clients
@EnableDiscoveryClient       // Register with Eureka service registry
public class MicroserviceClienteApplication {

    public static void main(String[] args) {
        SpringApplication.run(MicroserviceClienteApplication.class, args);
    }
}
```

<Note>
**Key Annotations:**
- `@SpringBootApplication`: Combines `@Configuration`, `@EnableAutoConfiguration`, and `@ComponentScan`
- `@EnableFeignClients`: Scans for `@FeignClient` interfaces
- `@EnableDiscoveryClient`: Enables service discovery via Eureka
</Note>

## Dependency Injection Patterns

### Constructor Injection (Recommended)

All services use **constructor-based dependency injection** for better testability and immutability:

```java microservice-cliente/src/main/java/com/microservice/cliente/service/ClienteService.java
@Slf4j
@Service
public class ClienteService {

    private final ClienteRepository clienteRepository;
    private final EntregaServiceClient entregaServiceClient;
    private final ClienteMapper clienteMapper;

    // Constructor injection - no @Autowired needed in Spring 4.3+
    public ClienteService(ClienteRepository clienteRepository, 
                         EntregaServiceClient entregaServiceClient,
                         ClienteMapper clienteMapper) {
        this.clienteRepository = clienteRepository;
        this.entregaServiceClient = entregaServiceClient;
        this.clienteMapper = clienteMapper;
    }

    // Service methods...
}
```

<Check>
**Benefits of Constructor Injection:**
- Fields can be `final`, ensuring immutability
- Dependencies are explicit and required
- Easier to test with mock objects
- No need for `@Autowired` annotation
</Check>

### Field Injection (Legacy Pattern)

```java
@RestController
@RequestMapping("/clientes")
public class ClienteController {

    @Autowired  // Legacy pattern - avoid in new code
    private ClienteService clienteService;
}
```

<Warning>
Field injection makes testing harder and hides dependencies. Use constructor injection instead.
</Warning>

## Service Layer Architecture

### Service Responsibilities

Services contain **business logic** and orchestrate operations:

```java microservice-cliente/src/main/java/com/microservice/cliente/service/ClienteService.java
@Slf4j
@Service
public class ClienteService {

    // 1. Simple CRUD operations
    public Cliente addCliente(Cliente cliente) {
        return clienteRepository.save(cliente);
    }

    // 2. Business logic with external service calls
    public List<ClienteDTO> getAllClientesConInfoRuta() {
        List<Cliente> clientes = clienteRepository.findAll();
        
        if(clientes.isEmpty()) {
            return Collections.emptyList();
        }

        List<Long> clienteIds = clientes.stream()
                .map(Cliente::getId)
                .collect(Collectors.toList());

        // Call external service via Feign
        Map<String, String> rutasPorCliente = obtenerNombresRutaBatch(clienteIds);

        // Map entities to DTOs
        return clientes.stream().map(
            cliente -> {
                String nombreRuta = rutasPorCliente.getOrDefault(
                    cliente.getId().toString(), 
                    "Sin ruta asignada"
                );
                return clienteMapper.toDTO(cliente, nombreRuta);
            }
        ).collect(Collectors.toList());
    }

    // 3. Error handling with custom exceptions
    @Transactional
    public void deleteCliente(Long id) {
        clienteRepository.findById(id)
            .orElseThrow(() -> new ClienteNotFoundException(id));
        
        try {
            entregaServiceClient.eliminarRelacionesCliente(id);
            clienteRepository.deleteById(id);
            log.info("Cliente con ID {} eliminado exitosamente", id);
        } catch (Exception e) {
            log.error("Error al eliminar cliente con ID {}: {}", id, e.getMessage(), e);
            throw new ClienteDeleteException(id, e);
        }
    }
}
```

<Tip>
Use `@Slf4j` (Lombok) for logging instead of manually creating logger instances.
</Tip>

## Controller Patterns

### REST Controller Structure

```java microservice-cliente/src/main/java/com/microservice/cliente/controller/ClienteController.java
@RestController
@RequestMapping("/clientes")
public class ClienteController {

    @Autowired
    private ClienteService clienteService;

    @PreAuthorize("hasRole('ADMIN')")
    @GetMapping()
    public List<ClienteDTO> getAllClientes() {
        return clienteService.getAllClientesConInfoRuta();
    }

    @PreAuthorize("hasRole('ADMIN')")
    @PostMapping()
    public Cliente addCliente(@RequestBody Cliente cliente) {
        return clienteService.addCliente(cliente);
    }

    @PreAuthorize("hasAnyRole('ADMIN', 'DRIVER')")
    @GetMapping("/{id}")
    public List<ClienteDTO> getClienteByIds(@PathVariable List<Long> id) {
        return clienteService.getClienteByIds(id);
    }

    @PreAuthorize("hasRole('ADMIN')")
    @DeleteMapping("/{id}")
    public ResponseEntity<String> deleteCliente(@PathVariable Long id) {
        clienteService.deleteCliente(id);
        return ResponseEntity.ok("Cliente eliminado exitosamente");
    }

    @PreAuthorize("hasRole('ADMIN')")
    @PutMapping("/{id}")
    public ResponseEntity<Cliente> updateCliente(
            @PathVariable Long id, 
            @RequestBody Cliente clienteDetails) {
        ResponseEntity<Cliente> updatedCliente = clienteService.updateCliente(id, clienteDetails);
        return ResponseEntity.ok(updatedCliente.getBody());
    }
}
```

### Controller Annotations

| Annotation | Purpose | Example |
|------------|---------|----------|
| `@RestController` | Combines `@Controller` + `@ResponseBody` | Class level |
| `@RequestMapping` | Base path for all endpoints | `@RequestMapping("/clientes")` |
| `@GetMapping` | HTTP GET endpoint | `@GetMapping("/{id}")` |
| `@PostMapping` | HTTP POST endpoint | `@PostMapping()` |
| `@PutMapping` | HTTP PUT endpoint | `@PutMapping("/{id}")` |
| `@DeleteMapping` | HTTP DELETE endpoint | `@DeleteMapping("/{id}")` |
| `@PathVariable` | Extract URL path parameter | `@PathVariable Long id` |
| `@RequestBody` | Bind request JSON to object | `@RequestBody Cliente cliente` |
| `@PreAuthorize` | Role-based authorization | `@PreAuthorize("hasRole('ADMIN')")` |

See [security.mdx](/development/security) for authorization details.

## Exception Handling

### Global Exception Handler

Use `@RestControllerAdvice` for centralized exception handling:

```java microservice-cliente/src/main/java/com/microservice/cliente/exception/GlobalExceptionHandler.java
@Slf4j
@RestControllerAdvice
public class GlobalExceptionHandler {

    @ExceptionHandler(ClienteNotFoundException.class)
    public ResponseEntity<Map<String, Object>> handleClienteNotFound(ClienteNotFoundException ex) {
        log.warn("Cliente no encontrado: {}", ex.getMessage());
        
        Map<String, Object> response = new HashMap<>();
        response.put("error", "Cliente no encontrado");
        response.put("message", ex.getMessage());
        response.put("status", HttpStatus.NOT_FOUND.value());
        
        return ResponseEntity.status(HttpStatus.NOT_FOUND).body(response);
    }

    @ExceptionHandler(ClienteDeleteException.class)
    public ResponseEntity<Map<String, Object>> handleClienteDeleteError(ClienteDeleteException ex) {
        log.error("Error al eliminar cliente: {}", ex.getMessage(), ex);
        
        Map<String, Object> response = new HashMap<>();
        response.put("error", "Error al eliminar cliente");
        response.put("message", ex.getMessage());
        response.put("status", HttpStatus.INTERNAL_SERVER_ERROR.value());
        
        return ResponseEntity.status(HttpStatus.INTERNAL_SERVER_ERROR).body(response);
    }

    @ExceptionHandler(Exception.class)
    public ResponseEntity<Map<String, Object>> handleGenericException(Exception ex) {
        log.error("Error inesperado: {}", ex.getMessage(), ex);
        
        Map<String, Object> response = new HashMap<>();
        response.put("error", "Error interno del servidor");
        response.put("message", ex.getMessage());
        response.put("status", HttpStatus.INTERNAL_SERVER_ERROR.value());
        
        return ResponseEntity.status(HttpStatus.INTERNAL_SERVER_ERROR).body(response);
    }
}
```

## Testing Strategies

### Unit Tests with JUnit 5 and Mockito

#### Service Layer Tests

```java microservice-cliente/src/test/java/com/microservice/cliente/service/ClienteServiceTest.java
@ExtendWith(MockitoExtension.class)
@DisplayName("ClienteService Tests")
class ClienteServiceTest {

    @Mock
    private ClienteRepository clienteRepository;

    @Mock
    private EntregaServiceClient entregaServiceClient;

    @InjectMocks
    private ClienteService clienteService;

    private Cliente clienteTest;

    @BeforeEach
    void setUp() {
        clienteTest = new Cliente();
        clienteTest.setId(1L);
        clienteTest.setNombre("Cliente Test");
        clienteTest.setEmail("test@test.com");
    }

    @Test
    @DisplayName("Debería agregar un cliente exitosamente")
    void testAddCliente_Success() {
        // Arrange
        when(clienteRepository.save(any(Cliente.class))).thenReturn(clienteTest);

        // Act
        Cliente resultado = clienteService.addCliente(clienteTest);

        // Assert
        assertNotNull(resultado);
        assertEquals(1L, resultado.getId());
        assertEquals("Cliente Test", resultado.getNombre());
        verify(clienteRepository).save(clienteTest);
    }

    @Test
    @DisplayName("Debería lanzar excepción cuando cliente no existe")
    void testDeleteCliente_NotFound() {
        // Arrange
        when(clienteRepository.findById(999L)).thenReturn(Optional.empty());

        // Act & Assert
        RuntimeException exception = assertThrows(RuntimeException.class, 
            () -> clienteService.deleteCliente(999L));
        
        assertTrue(exception.getMessage().contains("Cliente no encontrado"));
        verify(clienteRepository).findById(999L);
        verify(entregaServiceClient, never()).eliminarRelacionesCliente(anyLong());
    }
}
```

#### Controller Layer Tests

```java microservice-cliente/src/test/java/com/microservice/cliente/controller/ClienteControllerTest.java
@WebMvcTest(value = ClienteController.class, 
    excludeAutoConfiguration = {
        org.springframework.boot.autoconfigure.security.servlet.SecurityAutoConfiguration.class
    })
@AutoConfigureMockMvc(addFilters = false)
@DisplayName("ClienteController Tests")
class ClienteControllerTest {

    @Autowired
    private MockMvc mockMvc;

    @Autowired
    private ObjectMapper objectMapper;

    @MockBean
    private ClienteService clienteService;

    @Test
    @DisplayName("Debería obtener cliente por ID")
    void deberiaObtenerClientePorId() throws Exception {
        ClienteDTO clienteDTO = new ClienteDTO();
        clienteDTO.setId(1L);
        clienteDTO.setNombre("Cliente Test");
        
        when(clienteService.getClienteById(1L)).thenReturn(clienteDTO);

        mockMvc.perform(get("/clientes/cliente/1"))
                .andExpect(status().isOk())
                .andExpect(jsonPath("$.id").value(1))
                .andExpect(jsonPath("$.nombre").value("Cliente Test"));

        verify(clienteService).getClienteById(1L);
    }

    @Test
    @DisplayName("Debería crear cliente exitosamente")
    void deberiaCrearClienteExitosamente() throws Exception {
        Cliente cliente = new Cliente();
        cliente.setNombre("Nuevo Cliente");
        
        when(clienteService.addCliente(any(Cliente.class))).thenReturn(cliente);

        mockMvc.perform(post("/clientes")
                .contentType(MediaType.APPLICATION_JSON)
                .content(objectMapper.writeValueAsString(cliente)))
                .andExpect(status().isOk())
                .andExpect(jsonPath("$.nombre").value("Nuevo Cliente"));
    }
}
```

### Test Annotations Reference

| Annotation | Purpose |
|------------|----------|
| `@ExtendWith(MockitoExtension.class)` | Enable Mockito in JUnit 5 |
| `@WebMvcTest` | Test only web layer (controllers) |
| `@AutoConfigureMockMvc` | Configure MockMvc |
| `@Mock` | Create mock instance |
| `@MockBean` | Add mock to Spring context |
| `@InjectMocks` | Inject mocks into tested class |
| `@BeforeEach` | Run before each test |
| `@DisplayName` | Custom test description |
